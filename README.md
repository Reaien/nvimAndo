# nvimAndo

📦 Mi configuración personalizada de **Neovim + LazyVim**.  
Incluye mis plugins, keymaps y ajustes guardados en este repo para poder usarlos en cualquier máquina.

---

## 🚀 Instalación rápida

### 1. Clonar este repo

```bash
git clone https://github.com/BoyAndo/nvimAndo.git
cd nvimAndo

### 2. Ejecutar script
bash bootstrap.sh
```
